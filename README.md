# Expense Tracker

A simple and intuitive web application for tracking personal income and expenses. Built with vanilla JavaScript, HTML5, and CSS3.

## Features

- Add and remove income/expense transactions
- Automatic balance calculation
- Persistent data storage using localStorage
- Responsive design for mobile and desktop
- Currency formatting
- Visual distinction between income and expenses
- Real-time updates

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/expense-tracker.git
```

2. Navigate to the project directory:
```bash
cd expense-tracker
```

3. Open `index.html` in your preferred web browser

## Usage

1. Add a new transaction:
   - Enter a description for the transaction
   - Enter the amount (positive for income, negative for expenses)
   - Click "Add Transaction"

2. View your financial summary:
   - Total balance at the top
   - Income and expenses summary in the middle
   - Transaction history list below

3. Remove transactions:
   - Hover over any transaction
   - Click the "x" button that appears

## Features in Detail

- **Balance Container**: Shows current total balance
- **Income/Expense Summary**: Visual overview of total income and expenses
- **Transaction History**: Chronological list of all transactions
- **Form Controls**: Input fields for new transaction entries
- **Persistent Storage**: Data saves between sessions
- **Responsive Design**: Works on all device sizes
